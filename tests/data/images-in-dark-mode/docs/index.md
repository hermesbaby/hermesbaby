# Images in Dark Mode

Images shall not be affected by the dark mode. Always display (as it were) in light mode.

As we do it at GitHub.

```{drawio-figure} _figures/some_figure.drawio
some_figure.drawio
```

```{figure} _figures/some_figure.drawio.png
some_figure.drawio.png
```

```{figure} _figures/some_figure.drawio.svg
some_figure.drawio.svg
```
